## Comandos de proyecto estándar
- `/project:init` — Inicializar un proyecto nuevo desde template
- `/project:status` — Ver resumen del estado actual
- `/project:update-context` — Actualizar toda la documentación
- `/project:close-session` — Cerrar sesión y documentar
- `/project:suggest-tools` — Sugerir herramientas útiles
